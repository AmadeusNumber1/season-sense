# SeasonSense

A super simple, modern web app that shows current seasonal allergy (pollen) levels based on:

- your current geolocation, or
- a searched city.

## Run locally

From this folder:

```bash
python3 -m http.server 8080
```

Open: http://localhost:8080

## Data source

- Open-Meteo Air Quality API (pollen)
- Open-Meteo Geocoding API
